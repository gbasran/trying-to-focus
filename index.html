FILE_01 :: THE_DRIFT

In this simulation, the target constantly drifts away from input. This simulates Executive Dysfunction.

// OBSERVATION

Individuals with ADHD often identify the objective (the target) clearly, yet the attentional mechanism (the drift) refuses to lock on. This is a failure of regulation, not willpower.

Abandoning the target to click red nodes causes a drop in efficiency. This represents Switching Costs. For a neurotypical brain, switching contexts is low-cost. For an ADHD brain, the metabolic cost to refocus is significantly higher.

>> STATUS: FALSE

REALITY: It is a regulation disorder. It often manifests as "Inattentive" (internal noise, time blindness) rather than physical movement.

MYTH: "Everyone is a little ADHD."

REALITY: Everyone experiences distraction. ADHD is defined by frequency and severity that impairs life functions, employment, and health.

FILE_03 :: EXTERNAL_RAM

Internal working memory (RAM) is compromised. Survival requires "External RAM" protocols:

  • :: BODY_DOUBLING: Using another presence to anchor focus.
  • :: VISUAL_TIMERS: replacing abstract time with visual data (e.g., the bar in this sim).
